# SwiftRide
SwiftRide is a WhatsApp-first mobility and transaction system designed to run real transport and service payments using simple tools, real ledgers, and live status tracking.  SwiftRide allows riders, drivers, and operators to interact entirely through WhatsApp, while all money movement, records, and decisions are controlled through a central ledger
# SwiftRide

SwiftRide is a WhatsApp-first mobility and payments platform designed for real-world operations in emerging markets.

## Core Principles
- Phone-first operations
- WhatsApp as primary interface
- Ledger-based transactions
- No fake balances
- Real-time status tracking

## Current Stack
- WhatsApp Business
- Google Forms
- Google Sheets
- Google Apps Script
- Make.com
- GitHub (system source of truth)
